## How to add your own

Drop a markdown file into `src/content/blog/` — the file name becomes the URL.
Each file starts with a small frontmatter block:

```markdown
---
title: My Real Post
date: 2026-06-24
excerpt: One sentence shown in the blog list.
tags: [Java, Notes]
---

Your content here.
```

That's it — it shows up automatically on the next build. Delete this file when
you're ready.
